# Pinned dependency set verified to install and run the test suite together.
#
# These pins encode a few non-obvious compatibility constraints discovered while
# wiring up CI:
#   * numpy is held at 1.26.x: scikit-learn 1.3.2 (a finrl dependency) is built
#     against the numpy 1.x ABI and crashes on numpy 2.x.
#   * yfinance is held at 0.2.52: newer yfinance needs websockets>=13, which
#     conflicts with alpaca-trade-api (pulled in transitively by finrl) needing
#     websockets<11.
#   * alpaca-trade-api / exchange-calendars / stockstats / ta are explicit because
#     finrl imports them at package-import time.
